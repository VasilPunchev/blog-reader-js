# Blog Reader

A JavaScript application that dynamically loads blog posts and comments from a REST API.

🔗 Live Demo: https://vasilpunchev.github.io/blog-reader-js/

---

## 📌 Overview

This project demonstrates asynchronous JavaScript, REST API consumption, and dynamic DOM manipulation.

The application allows users to:
- Load all available blog posts
- View selected post details
- Filter and display related comments
- Interact with dynamically rendered UI content

---

## 🚀 Features

- Fetch API
- Async/Await
- Dynamic DOM updates
- Data filtering by postId
- Event-driven programming
- Clean modular structure

---

## 🛠 Technologies Used

- JavaScript (ES6+)
- HTML5
- CSS3
- REST API
- Git & GitHub

---

## ⚠️ Important Note

The project requires a local REST server (`localhost:3030`) to function fully.  
The live demo shows the interface, but API requests require a running backend server.
